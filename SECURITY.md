# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public issue.**

Instead, please email **yumi@yumi.to** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You can expect an acknowledgment within 48 hours and a resolution timeline within 7 days.

## Scope

This policy applies to the static site hosted at [ship.yumi.to](https://ship.yumi.to) and its source code in this repository.
