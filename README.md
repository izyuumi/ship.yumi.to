## Stack

- Static HTML + [Tailwind CSS](https://tailwindcss.com) (CDN)
- Hosted on [GitHub Pages](https://pages.github.com)
- No build step, no framework, no dependencies

## Adding a New Page

1. Create a directory: `mkdir my-product`
2. Add `index.html` with Tailwind CDN
3. Run `./generate-sitemap.sh` to update the sitemap
4. Push to `main` â€” GitHub Pages deploys automatically
