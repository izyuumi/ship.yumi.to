How HourProof works

Log shifts in seconds. Map hours to grants. Export a clean report.

👥
Step 1

Set up your roster

Add volunteers by name and type — board, staff, intern, general. Import from CSV in seconds.

📅
Step 2

Log shifts

Quick-log: pick volunteer, enter date + hours, select which program. Done in 15 seconds per shift. Or let volunteers self-log via a simple link.

Map to grants

Connect programs to grants. One program can count toward multiple grants. Set required match hours per grant.
